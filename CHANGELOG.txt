Version: 1.0.0  
Date: 26.08.2024

  Major Features:
    - arithmetic, logical, bitwise, and comparison operations (+, -, *, /, &&, ||, >>, ==, >, etc)
    - variable declarations and assignments
    - array declarations
    - support for pointers and dereferencement
    - support for 64-bit integer variables only
    - function declarations with parameter passing and return statements
    - conditional statements (if, else)
    - loop constructs (loop, break)
    - support for annotations using '!' syntax
    - support for macros (#def, #ifdef, #ifndef, #else, #endif)
    - standard library with support for IO operations, memory management, math, etc
